Scaling Rule Types

Time-Based Scaling:
* Business hours patterns
* Weekend vs weekday schedules
* Holiday schedules
* Timezone-aware scheduling

Metric-Based Scaling:
* Standard Kubernetes metrics (CPU, memory)
* Custom application metrics (requests per second, queue length)
* External business metrics (sales volume, user activity)
* Composite metrics (combining multiple sources)

Event-Based Scaling:
* Calendar events
* External webhook triggers
* Manual override capabilities
